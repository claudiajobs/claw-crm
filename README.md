# CLAW — Monorepo

## Apps

| App | Descrição | Stack |
|---|---|---|
| `apps/crm` | CRM para fábrica de tintas | Next.js 16 + Supabase + Vercel |

## Setup

```bash
# Instalar dependências do app CRM
cd apps/crm && npm install

# Rodar em dev
npm run crm
```

## Estrutura

```
apps/
  crm/          ← CRM (Next.js)
  sdr-whatsapp/ ← SDR WhatsApp (futuro)
  sdr-instagram/← SDR Instagram (futuro)
```
